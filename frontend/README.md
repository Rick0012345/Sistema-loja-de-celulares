# Frontend

Painel Next.js do sistema de gestão da loja de celulares.

## Tela inicial entregue

- cards com indicadores do dashboard
- lista de ordens recentes
- alerta de estoque baixo
- formulário para cadastro de clientes
- formulário para cadastro de produtos
- formulário para abertura de ordens de serviço

## Execução

Na raiz do projeto:

```bash
docker compose --profile scaffold up -d frontend
```
